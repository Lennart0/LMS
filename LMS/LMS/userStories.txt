---- Lärare -----

User Story 1:
Som Ej Inloggad registererad lärare
kan läraren loggar in i LMS, med användarnamn och lösenord
för att uppnå inloggning i systemet och komma till Kurslistning-vyn.

User Story 2: 
Som Inloggad Lärare
kan jag navigera till (via navigerinsbar) Kurslistning-vyn
för att kunna se alla kurser. 


User Story 3: 
Som lärare
kan jag klicka på specifik kurs i kurslistning-vyn
för att komma till kursdealj-vyn för kursen för att se kursens samtliga egenskaper, relaterade moduler och inlämningsuppgifter.

User Story 4:
Som lärare
kan jag klicka på specifik modul i Kursdetelj-vyn
för att komma till moduldetalj-vyn där Aktiviteter listas.

User Story 5: 
Som Lärare i kursdetaljvy
kan jag klicka på inlämnings-uppgift
för att se samtiga elevinlämningar tillhörande uppgiften.

User Story 6:
Som Lärare
kan jag klicka på länk i kurslistnings-vyn
för att registera ny lärare.

User Story 7: 
Som Lärare
kan klicka på länk i kursdetalj-vyn
för att registera ny elev.

User Story 8:
Som lärare
kan i kurslistnings-vyn klicka på knapp (eller liknande)
för att skapa ny kurs och därmed också komma till kursdetalj-vyn (editeringsläge) för den nya kursen.

User Story 9:
Som lärare
kan jag i Kurslistnings-vyb eller kursdetalj-vyn genom att 'klicka'
komma till kursdetalj-vyn i editeringläge och därmed kunna modifiera kursen.

User Story 10: 
Som lärare
kan jag i kursdetalj-vyn i editeringläge, 'klicka'
för att skapa en ny modul och därmed komma till moduldetalj-vyn (editeringsläge) för den nya modulen.

User Story 11:
Som lärare 
kan jag från modullistning (i kursdetalj-vyn) eller moduldetalj-vyn, 'klicka'
för att hamna i moduldetalj-vyn i editeringsläge för att kunna modifiera modulen.

User Story 12: 
Som lärare
kan jag i kursdetalj-vyn i editeringläge, 'klicka'
för att ladda up dokument assosierade med kursen.

User Story 13: 
Som lärare
kan jag i moduldetalj-vyn i editeringsläge, 'klicka'
för att ladda up dokument assosierade med modulen.

User Story 14: 
Som lärare 
kan i modulschemat 'klicka'
för att skapa ny aktivitet som ska ingå i schemat.

User Story 15: 
Som lärare
kan jag klicka på specifik aktivitet i modulschemat
för att editera aktiviteten.

User Story 16: 
Som lärare
kan jag klicka i moduleschemat(wireframe#3) (editeringsläge)
för att ladda up övningar(filer) med begränsad inlämnings tid.

------Elev----------

User Story 20:
Som Ej Inloggad, registererad elev
kan jag logga in i LMS med användarnamn och lösenord
för att bli inloggad och komma till Modulschemat.

User Story 21: 
Som elev
kan jag navigera till (via navigerinsbar) kursöversikts-vyn(Wireframe#1) (för elever)
för att där se klasskamrater, kursinfo och dokument för kursen.

User Story 22:
Som elev
kan jag navigera (via navigerinsbar)
för att komma till modulschemat där moduler och aktiviterer listas med datum och ikon om filer är assosierade.

User story 23:
Som elev
kan jag navigera (via navigerinsbar) till inlämningsuppgift-vyn
för att se samtliga inlämnings uppgifer och dess status 

User Story 24: 
Som elev
kan jag klicka på aktivitet i modulschemat
för att komma till aktivitetsdetalj-vyn.

User Story 25: 
Som elev
kan jag klicka i aktivitetsdetalj-vyn
för att ladda upp uppgifts inlämningar.

user story 26:
Som elev
kan jag klicka på specifik inlämningsuppgift i inlämningsuppgift-vyn
för att kunna ladda up en uppgift för inlämning.

User Story 27: 
Som elev
kan jag klicka på kurs-associerat dokument i kursöversikts-vyn(Wireframe#1)
för att ladda ner detta dokument.

User Story 28: 
Som elev
kan jag klicka på modul-associerat dokument i modulschemat
för att ladda ner detta dokument.

User Story 29: 
Som elev
kan jag klicka på aktivitet-associerat dokument i aktivitetsdetalj-vyn
för att ladda ner detta dokument.
